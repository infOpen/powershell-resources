SHORT DESCRIPTION
    This module contains functions to manage JSON data.

LONG DESCRIPTION
    This module contains high level function to be used in scripts.
    Functions are in separeted files, documented and with a 100% code coverage.

EXAMPLES
    Load module :
        Import-Module Infopen.JSON

    Load JSON file content :
        Import-FromJsonFile -Path 'C:\good.json'
